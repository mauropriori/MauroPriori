### 🌍 Questo Repository

Questo repo ospita il codice sorgente del mio sito personale e portfolio, visitabile online.
Il sito è progettato con un'architettura **completamente statica ed estremamente ottimizzata**, senza dipendenze backend non necessarie.

* **Stack Principale:** HTML5 semantico, JavaScript Vanilla, Tailwind CSS.
* **Infrastruttura:** Containerizzato tramite **Docker** utilizzando un leggerissimo web server `BusyBox`.
* **Ottimizzazioni:** Riduzione dei blocchi render-blocking, utilizzo di file esterni minificati e conversione asset grafici in formato WebP per le massime performance.
* **Strumenti:** Sviluppato con il supporto di _Google Antigravity e Gemini 3_.
